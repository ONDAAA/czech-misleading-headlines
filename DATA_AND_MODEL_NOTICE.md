## Not automatically relicensed by MIT

The MIT license should not be assumed to automatically relicense:
- scraped headline corpora and source-derived text data

- third-party base models, tokenizers, or upstream model families
- generated model weights when their downstream use may depend on upstream model terms
- any third-party assets or material reproduced from external sources

## Practical interpretation

If you want to reuse code from this repository, the MIT license is the intended license.

If you want to reuse datasets, source-derived corpora, model artifacts, or other non-code materials, you should independently verify:
- copyright status
- database rights
- website terms of use
- upstream model license terms
- any institutional or publication constraints

When in doubt, treat non-code assets as requiring separate review before redistribution or commercial reuse.
